# Metro App

A full-stack application for calculating the shortest path between metro stations using C++ for the core algorithm and Node.js for the backend API, with a beautiful interactive frontend.

## Project Structure
- `backend/`: Node.js Express server that runs the compiled C++ metro logic.
- `frontend/`: Vanilla JS, CSS, and HTML user interface for the Metro App.

## Getting Started
1. **Compile C++ Logic**: Run `g++ metro.cpp -o metro` in the `backend/` directory.
2. **Setup Backend**: Run `npm install` in the `backend/` directory, then start the server using `node server.js`.
3. **Open Frontend**: Serve the `frontend/` directory or open `index.html` in your browser.
